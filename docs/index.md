# 🌟 Welcome to Northstar Developer Docs

**Northstar** is a modern, high-performance Configuration Management Database (CMDB) platform developed by **Astrona**. It delivers physical asset visualization, 2D CAD blueprint room designers, multi-floor server cabinet layouts, active discovery subnet scanners, and declarative GitOps reconciliation loops.

This documentation conforms strictly to the **Diátaxis framework** to support logical, goal-oriented technical learning and references.
